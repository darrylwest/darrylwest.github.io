Secret Message Cryptor Help

What does this thing do?

The cryptor App is a desktop/laptop browser application that takes a text message message that you enter and encrypts it based on your pass phrase. It then lets you download the results as an HTML file with the encrypted data. From there you can render the file in a browser, and decrypt the text using your pass phrase. The typical use case is securly saving a list of hard to remember passwords.

The file cannot be decrypted without the pass phrase, so it's important to create a phrase that is 1) long enough to be difficult to guess 2) easy to remember

Is it secure?

This is a browser application, with no personal data, text or pass phrase going to the server. The encryped file is only downloaded to your personal device, in your downloads folder. The pass phrase never leaves your browser and is not stored in cookies, local storage, etc.

Pass Phrase Examples

  • he who is brave is free. no?
  • you can totally do this.
  • try a slow count to 10, then speak.
  • choose one thing and go all in. or not.
  • I can, and I Will. Smith.

Steps

  1. Enter or copy/paste your message, list of passwords, etc to the message box
  2. Enter your pass phrase
  3. Click the Encrypt Message button
  4. Click the Download link
  5. Open the downloaded file, enter your pass phrase and verity the plain text content
  6. (Optional) Move the file to your desktop or other location with an alternate name.

You can then just leave the file in your downloads folder, or move it to an alternate, or even your desk top. Without the pass phrase, it can't be decrypted. And, the pass phrase never gets shared to any browser or other application.

Access and Decrypt

Just double click on the file to have it open in your default browser. Once open, enter the pass phrase and click the Decrypt button. The decrypted text will appear and the password and button will go away. Close or refresh the browser page to hide the plain text. The page will automatically time-out and close after 5 minutes.

Updating Existing File

You can update an existing file by first decrpting it, then copy/past to the a new application session. Make changes to the text, then re-encrypt and download the new file.

What is the number in filename?

The filename secret-####.html is created in the browser using the current date as a timestamp. The timestamp is from javascript's Date.getTime() method.
