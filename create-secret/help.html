Secret Message Cryptor Help

What does this thing do?

The Cryptor app is a desktop/laptop browser application that takes a text message you enter and encrypts it based on your passphrase. It then lets you download the result to your personal device (desktop or laptop) as an HTML file embedded with the encrypted data. From there, you can double-click the downloaded file to render it in a browser and decrypt the text using the same passphrase.

The typical use case is securly saving a list of hard to remember passwords.

The file cannot be decrypted without the passphrase, so it's important to create a phrase that is long enough to be difficult to guess, and easy to remember.

Is it secure?

This is a browser application, with no personal data, text, or passphrase going to the server or over the wire. The encrypted file is only downloaded to your personal device, typically in your downloads folder. The passphrase never leaves your browser and is not stored in cookies, local storage, etc. As with all internet applications that prompt for a password or passphrase, you should always check the address bar to verify the domain.

Steps

  1. Enter or copy/paste your message, list of passwords, etc to the message box
  2. Enter your passphrase
  3. Click the Encrypt Message button
  4. Click the Download link
  5. Open the downloaded file, enter your passphrase and verity the plain text content
  6. (Optional) Move the file to your desktop or other location with an alternate name.

Don't forget to close the browser page after verifying the encryption, especially if you are using this in a public setting, like when you have you laptop at a caf√©.

You can then just leave the file in your downloads folder, or move it to an alternate, or even your desk top. Without the passphrase, it can't be decrypted. And, the passphrase never gets shared to any browser or other application.

Access and Decrypt

Just double click on the file to have it open in your default browser. Once open, enter the passphrase and click the Decrypt button. The decrypted text will appear and the password and button will disappear. Close or refresh the browser page to hide the plain text. The page will automatically time-out and close after 5 minutes.

Updating Existing File

You can update an existing file by first decrpting it, then copy/past to the a new application session. Make changes to the text, then re-encrypt and download the new file.

What is the number in filename?

The filename secret-####.html is created in the browser using the current date as a timestamp. The timestamp is from javascript's Date.getTime() method.

What happens when I click Encrpt Message Button?

The passphrase is used to encrypt the message right in the browser (not a server action). The resulting cipher text is in the page and appended to the secrets file when you click the download link.
